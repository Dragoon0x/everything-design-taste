# The Shortform Guide to Everything Design Taste

## What This Is

A taste system for AI agents. Skills, agents, rules, and hooks that turn generic AI output into work with genuine design quality.

Think of it as a design director that never sleeps, never forgets the brand guidelines, and catches every instance of "leverage" in your copy.

## Quick Start

```bash
git clone https://github.com/0xDragoon/everything-design-taste.git
cd everything-design-taste
npm install
./install.sh --profile standard
```

## The Philosophy

1. **Design is direction, not decoration.** Every visual choice is a strategic choice.
2. **AI produces slop by default.** This system fights it with specific, opinionated rules.
3. **Every element earns its place.** No defaults. No "because that's standard."
4. **Writing quality is design quality.** Bad copy ruins good design every time.
5. **Taste is a skill.** It can be learned, encoded, and systematized.

## What's Inside

- **10 agents** — Specialized reviewers for design, brand, product, copy, visual QA, and more
- **22 skills** — Domain knowledge from typography to pitch decks to competitive teardowns
- **8 rules** — Always-on quality gates for taste, hierarchy, anti-slop, writing, and accessibility
- **4 hooks** — Automated checks for AI tone, contrast, consistency, and naming
- **5 design templates** — Ready-to-use design systems for different aesthetics
- **4 contexts** — Mode switching for product thinking, brand review, critique, and founder mode

## The One Rule

Before any output ships, ask: *Would a senior design director look at this and think it was made by someone who cares?*

If yes, ship it. If no, keep working.
