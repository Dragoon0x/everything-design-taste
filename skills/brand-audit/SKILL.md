# Brand Audit

## What to Audit

### Visual Identity
- [ ] Logo usage consistent across all touchpoints
- [ ] Color palette applied correctly (check hex values)
- [ ] Typography matches brand guidelines
- [ ] Photography/illustration style consistent
- [ ] Iconography from a single system

### Verbal Identity
- [ ] Voice attributes reflected in all copy
- [ ] Tone appropriate per context
- [ ] Terminology consistent (no conflicting names)
- [ ] Tagline/messaging aligned

### Touchpoints
| Touchpoint | Check |
|------------|-------|
| Website | Visual identity, voice, messaging |
| App | Component consistency, interaction patterns |
| Email | Template design, voice, CTA style |
| Social media | Visual consistency, voice |
| Support | Response tone, helpfulness |
| Sales materials | Messaging alignment, visual quality |
| Packaging | Physical brand expression |

## Scoring
- Consistent: matches guidelines exactly
- Drifting: close but deviating
- Inconsistent: clearly off-brand
- Missing: no guidelines exist for this touchpoint

## Output
Audit report with:
- Overall brand consistency score
- Per-touchpoint assessment
- Photographic evidence of issues
- Prioritized recommendations
