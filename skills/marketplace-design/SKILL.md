# Marketplace Design

## Marketplace Patterns

### Listing Page
- High-quality images (minimum standards enforced).
- Title, price, seller info, condition/details.
- Location/shipping information.
- Seller rating and review count.
- Contact/buy action prominent.

### Search & Discovery
- Category browsing + search.
- Filters: price range, location, condition, rating.
- Sort: relevance, price, newest, rating.
- Map view for location-based listings.

### Trust Systems
- Verified seller badges.
- Rating and review system (both sides rate each other).
- Secure payment/escrow.
- Dispute resolution process.
- Response time and rate indicators.

### Dual Experience
- Buyer: browse, filter, contact, purchase, review.
- Seller: list, manage, respond, ship, get paid.
- Dashboard for each role with relevant metrics.
