# Copywriting Frameworks

## Frameworks

### AIDA
- **Attention:** Hook that stops the scroll.
- **Interest:** Details that build curiosity.
- **Desire:** Benefits that create want.
- **Action:** Clear CTA.

### PAS
- **Problem:** Name the pain.
- **Agitate:** Make the pain vivid.
- **Solution:** Present the relief.

### BAB
- **Before:** Current state (pain).
- **After:** Desired state (pleasure).
- **Bridge:** How your product gets them there.

### FAB
- **Feature:** What it is.
- **Advantage:** What it does.
- **Benefit:** Why it matters to the user.

## When to Use Each
| Framework | Best For |
|-----------|----------|
| AIDA | Landing pages, ads |
| PAS | Email, social posts |
| BAB | Case studies, testimonials |
| FAB | Feature descriptions, product pages |
