# Event & Ticketing Design

## Event Discovery
- Visual calendar or list view
- Filter: date, category, location, price range
- Featured/promoted events section
- Personalized recommendations
- Save/bookmark events
- Share event with friends

## Event Detail Page
- Hero image or event banner
- Title, date/time, venue, organizer
- Clear pricing (ticket types and ranges)
- Seat map (if seated venue)
- Description, lineup, schedule
- Venue info with map
- "Add to Calendar" button
- Social proof (X attending, friends going)

## Ticket Selection
- Ticket types: name, price, description, availability
- Quantity selector with per-type limits
- Seat map: interactive, color-coded by price/section
- Show view from seat (if available)
- Accessibility seating options visible
- Timer for reservation hold (10-15 minutes)

## Checkout
- Order summary with ticket details
- Attendee info collection (per ticket if required)
- Payment processing
- Confirmation with QR code / barcode
- Email confirmation with calendar attachment
- Digital wallet pass option (Apple Wallet, Google Wallet)
