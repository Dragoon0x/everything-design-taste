# Food Delivery Design

## Restaurant Discovery
- Nearby restaurants sorted by: relevance, rating, delivery time, price.
- Cuisine filters, dietary preference filters.
- Estimated delivery time + delivery fee shown on cards.
- Promoted listings clearly labeled.

## Menu Browsing
- Categories with item counts.
- Item: photo + name + description + price + dietary tags.
- Popular/recommended items highlighted.
- Customization: modifiers, add-ons, special instructions.

## Cart & Checkout
- Persistent cart accessible from any page.
- Item count badge on cart icon.
- Delivery address + instructions.
- Tip selection (preset amounts + custom).
- Order summary: items + subtotal + fees + tip + total.

## Order Tracking
- Real-time status: Confirmed → Preparing → Picked Up → Arriving.
- Map with driver location.
- Estimated arrival countdown.
- Contact driver option.
- Rate and tip after delivery.
