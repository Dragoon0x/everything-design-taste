# Thank You Page Design

## Structure
1. Confirmation: "You're in" / "Order confirmed" / "Payment received"
2. Details: Order number, email confirmation mention, timeline
3. Next step: One clear action (download, get started, share)
4. Optional: related content, referral offer, social share

## Rules
1. Don't just say "Thank you" — give them something to do
2. Set expectations: what happens next and when
3. Provide reference info (order number, confirmation email)
4. This is a high-engagement moment: use it wisely
5. Track conversions here (this is your conversion event)
