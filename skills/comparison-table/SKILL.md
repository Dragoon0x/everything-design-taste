# Comparison Table

## Design
- Sticky header row with plan/product names
- Group features by category with section headers
- Checkmarks and X marks (with accessible labels)
- Highlight the recommended column
- Mobile: convert to stacked cards or horizontal scroll

## Rules
1. Highlight differences, not similarities
2. Most important features at the top
3. Tooltip on features that need explanation
4. 10-15 features max before it becomes overwhelming
5. CTA button at the bottom of each column
6. Include price at top and bottom for context
