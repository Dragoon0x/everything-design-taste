# Onboarding Email Sequence

## Sequence Structure
| Day | Email | Goal |
|-----|-------|------|
| 0 | Welcome | Confirm signup, set expectations |
| 1 | Quick win | Guide to first valuable action |
| 3 | Feature highlight | Show a key feature they haven't tried |
| 7 | Social proof | Customer story, use case inspiration |
| 14 | Check-in | Ask if they need help, offer support |

## Rules
1. One CTA per email
2. Skip emails if user already completed the action
3. Stop sequence if user converts to paid
4. Subject lines: benefit-focused, under 50 characters
5. Frequency: never more than one per day
6. Always include unsubscribe
