# Design Portfolio Building

## Portfolio Strategy

### Curation
- 3-5 strong case studies. Quality beats quantity.
- Show range: different project types, industries, or skills.
- Lead with your strongest piece.
- Remove anything you can't talk about confidently.

### Case Study Structure
1. Hero image/video (the best visual from the project).
2. One-line project summary.
3. Role + timeline + team.
4. Problem + process + solution + impact.
5. 1,500-2,500 words. Tight editing.

### What Reviewers Look For
- **Thinking:** How you approach problems (not just the output).
- **Craft:** Execution quality and attention to detail.
- **Impact:** Did it work? How do you know?
- **Communication:** Can you explain your decisions clearly?
- **Self-awareness:** What would you do differently?

### Portfolio Site Design
- The site itself is a portfolio piece. Treat it accordingly.
- Fast loading (nobody waits for a designer's slow site).
- Mobile-friendly.
- Clear navigation between projects.
- Contact info easy to find.
- No password protection (hiring managers won't request access).
