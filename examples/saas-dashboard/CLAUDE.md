# SaaS Dashboard - Project Config

## Product Context
A B2B analytics dashboard for mid-market companies. Data-dense, professional, functional.

## Design Direction
- Template: developer-tool (dark, dense, precise)
- Priority: Information density over visual flair
- Users: Data analysts and operations managers (power users)

## Active Skills
anti-slop, design-system-patterns, typography-craft, color-systems, responsive-craft, design-spec-writing

## Active Agents
design-reviewer, visual-qa, design-system-auditor

## Active Rules
common/*, react-ui/*

## Key Decisions
- Dark mode default (users stare at this 8 hours a day)
- Tables over cards for data display
- Keyboard shortcuts for power users
- Density toggle (compact/comfortable/spacious)
